The Stygian Library Generator

An automatic generator that follows the procedures and random tables detailed in The Stygian Library by Emmy "Cavegirl" Allen.

To use the generator, click "Go deeper" to advance from the selected room. Select a different room to advance from that room instead. Click "Event" to generate an event appropriate for the selected room. Some rooms have conditions that affect which events are generated.

Toggle between resting and exploring PCs by clicking the button named "PCs are exploring/resting". Toggle whether the PCs are considered visitors or intruders by clicking "PCs are visitors/intruders". These toggles both affects the types of events that are generated.

Add manual connections between rooms or manually delete connections (or rooms) with the "Add connection" and "Delete selection" buttons.

Finally, save and load your generated library with the "Save/Load" buttons.

This project is on GitHub, feel free to contribute!
